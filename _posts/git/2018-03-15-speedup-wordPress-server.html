Adding more changes to your last commit

  1. The Basic of the Amend Command

    {% highlight Bash shell scripts %} $ (some_branch) git add changelog.md {% endhighlight %}

    And amend it:

    {% highlight Bash shell scripts %} $ (some_branch) git commit --amend {% endhighlight %}

    After executing the command, this file will show up vi/vim. check this cheatsheet if you don't know how to use it.

  2. Amending a Commit Without Changing Its Message

    If you donâ€™t want to change your commit message, you can run the amend command with the no-edit flag, like this:

    {% highlight Bash shell scripts %} $ (some_branch) git commit --amend --no-edit {% endhighlight %}

  3. Just Editing a Commit Message

    {% highlight Bash shell scripts %} $ (some_branch) git commit --amend -m "Your new commit message" {% endhighlight %}

  4. Pushing an Amended Commit

    {% highlight Bash shell scripts %} $ (some_branch) git push -f origin some_branch {% endhighlight %}
